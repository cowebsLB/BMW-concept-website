# Feature Breakdown

## Landing Experience
- Cinematic hero with generated/fallback background image
- Scroll-driven motion and reveal animations
- Branded typography and immersive visual composition

## Navigation
- Fixed top navigation with responsive behavior
- Slide-in menu for mobile and compact browsing
- Command palette with keyboard shortcut for fast navigation

## Core Sections
- Showroom
- Models
- Electric
- Configurator
- Innovation
- Shop
- Test Drive

Each section is routed through React Router and rendered as a dedicated page component.

## Shared State
- Experience context tracks interaction events and lightweight user preference snapshots (favorites).
