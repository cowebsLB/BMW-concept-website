# BMW Concept Website

## Project Overview
BMW Vision Next is a cinematic React single-page experience that showcases concept vehicles, electric innovation, a showroom flow, and brand storytelling.

## Features
- Immersive landing hero with animated transitions
- Multi-page experience using React Router
- Showroom, models, electric, configurator, innovation, shop, and test drive pages
- Context-based interaction tracking and favorites snapshot
- Responsive navigation and command palette

## Setup Instructions
1. Install dependencies:
   - `npm install`
2. Start local development:
   - `npm run dev`
3. Create a production build:
   - `npm run build`
4. Preview the production build:
   - `npm run preview`

## Usage
- Open the app in a browser and navigate through sections using the menu.
- Use `Ctrl + K` (or `Cmd + K`) to open the command palette.
- For deployment to GitHub Pages project sites, use the production build output from `dist/`.
